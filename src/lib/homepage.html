Type the URL of your dreams into the location bar and hit enter, the LLM will tell you want it thinks the website will look like. The weirder the better.

Examples

  • twitter.com/@christophercolumbus
  • twitter.com/@shakespeare
  • geocities.com/ResearchTriangle/1979/
  • winamp.com
  • greatestpoems.com
  • cutest-llamas.org

The way this hack works is by taking the URL and getting the LLM to generate the HTML for the URL.

Along the way, it will fan out and also generate approximation for images that might be contained, and optionally any Javascript that could be running on the page.

For the HTML generation, some amount of prompt refinement was needed to make the LLM generate the HTML in a way that could be rendered in a browser. Subsequently, I found it easier to use Mistral Vibe to iterate on the prompt and directly edit the prompt after explaining to it what I wanted.
